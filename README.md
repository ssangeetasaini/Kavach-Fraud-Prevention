# Kavach-Fraud-Prevention# Kavach: AI-Powered Fraud Prevention Tool

"Kavach" is an AI-driven accessibility tool specifically designed to protect non-literate users from financial fraud. The system performs real-time monitoring of incoming SMS and calls, triggering instant audio alerts in regional languages upon detecting suspicious activity.

## Key Features
- **Real-time Monitoring:** Constant tracking of SMS and incoming calls to identify potential threats.
- **AI-Powered Detection:** Utilizes Machine Learning classifiers to detect malicious URLs and complex phishing patterns.
- **Accessibility-First Design:** Implements Text-to-Speech (TTS) technology to provide immediate voice warnings in regional languages, ensuring ease of use for non-literate individuals.

## Tech Stack
- **Language:** Python
- **Libraries:**  scikit-learn, gTTS, pandas
- **Platform:** Cmd Prompt

## Development Context
This project was developed during my internship at the Police Headquarters (PHQ) Cyber Cell. It was engineered by studying and analyzing real-world financial fraud patterns reported via the 1930 Cyber Crime Helpline.
